## About this framework

このフレームワークはLaravelを使用しており、さまざまなWEBアプリケーションを作成するときの骨組みとなります。
WEBアプリケーション開発に必要なものを、あらかじめパッケージ化しています。

## How to use the framework

Publicフォルダにプログラムを設置し、サイトにアクセスするとインストールプログラムが立ち上がります。
